Primeiros Computadores

Os primeiros computadores foram máquinas enormes e primitivas que surgiram nas décadas de 1940 e 1950. Eles eram diferentes dos computadores modernos em tamanho, velocidade e capacidade de processamento. Os primeiros computadores eram baseados em válvulas e usavam sistemas de cartões perfurados para entrada e saída de dados. Esses computadores ocupavam salas inteiras e consumiam uma quantidade imensa de energia. Um exemplo de um dos primeiros computadores é o ENIAC (Electronic Numerical Integrator and Computer), que foi construído nos Estados Unidos durante a Segunda Guerra Mundial. O ENIAC era um monstro em termos de tamanho e peso, ocupando várias salas e pesando cerca de 30 toneladas. Ele era usado principalmente para cálculos científicos e militares. Outro exemplo importante é o UNIVAC I (Universal Automatic Computer), que foi o primeiro computador comercialmente disponível nos Estados Unidos. Ele foi desenvolvido pela empresa Remington Rand e foi utilizado para tarefas como previsão do tempo e processamento de dados censitários. Esses primeiros computadores eram muito limitados em comparação com os computadores atuais. Eles tinham capacidade de armazenamento e velocidade de processamento extremamente baixas em comparação com os dispositivos modernos. No entanto, essas máquinas pioneiras abriram caminho para o desenvolvimento da tecnologia da computação e pavimentaram o caminho para os computadores que usamos hoje em dia.

Origem Internet

(Voltar ao índice)

A internet é uma rede global de computadores interconectados que permite a troca de informações e comunicação em todo o mundo. Ela surgiu a partir de pesquisas e desenvolvimentos realizados por várias instituições e indivíduos ao longo do tempo. No final da década de 1960, o Departamento de Defesa dos Estados Unidos iniciou um projeto chamado ARPANET (Advanced Research Projects Agency Network) para criar uma rede de computadores resiliente e descentralizada, capaz de resistir a ataques e falhas. O objetivo principal era compartilhar recursos e informações entre universidades e centros de pesquisa. A ARPANET usava o protocolo de comunicação TCP/IP (Transmission Control Protocol/Internet Protocol), que se tornou a base da internet moderna. O TCP/IP permite que os computadores se comuniquem e troquem dados de forma eficiente e confiável. À medida que a ARPANET crescia, outras redes começaram a ser desenvolvidas em diferentes países e setores, como o MILNET para fins militares nos EUA e a JANET no Reino Unido para conectar instituições acadêmicas. No início dos anos 1990, a World Wide Web (WWW) foi criada por Tim Berners-Lee, um cientista da computação britânico, como uma maneira de organizar e compartilhar informações na internet. A WWW introduziu o conceito de hipertexto, permitindo que os usuários navegassem e acessassem informações de forma fácil e intuitiva por meio de links. Com o tempo, a internet se expandiu rapidamente e se tornou amplamente acessível para o público em geral. Novos serviços e tecnologias, como e-mail, navegadores da web, motores de busca e mídias sociais, foram desenvolvidos para melhorar a experiência dos usuários e ampliar ainda mais as possibilidades da internet. Hoje em dia, a internet desempenha um papel fundamental em diversas áreas, como comunicação, comércio, entretenimento, educação e pesquisa, conectando bilhões de pessoas ao redor do mundo e transformando a maneira como interagimos e compartilhamos informações.

Como funciona o HTML

(Voltar ao índice)

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar e estruturar páginas da web. Ele fornece uma estrutura básica para o conteúdo de um site, permitindo que os navegadores interpretem e exibam informações de forma adequada aos usuários. O HTML é composto por elementos, que são representados por tags, cercadas por colchetes angulares . Cada elemento descreve uma parte específica do conteúdo, como cabeçalhos, parágrafos, imagens, links e tabelas. Os elementos HTML são organizados em uma estrutura hierárquica, com um elemento principal chamado de "html" que envolve todo o conteúdo da página. Dentro do "html", há elementos de cabeçalho "head" que contêm informações sobre o documento, como o título da página e as referências a arquivos externos, como folhas de estilo CSS. O corpo do documento é definido pelo elemento "body", onde a maior parte do conteúdo visível é inserida. Dentro do "body", podem ser utilizados elementos como "h1" a "h6" para cabeçalhos, "p" para parágrafos, "img" para inserir imagens e "a" para criar links. Além dos elementos básicos, o HTML permite a aplicação de atributos aos elementos para fornecer informações adicionais. Os atributos especificam propriedades ou comportamentos dos elementos e são definidos dentro das tags de abertura. Por exemplo, o atributo href é usado para definir o destino de um link "a" e o atributo src especifica o caminho da imagem em um elemento "img". Os navegadores interpretam o código HTML e renderizam o conteúdo de acordo com as tags e atributos definidos. Isso significa que o HTML determina como os elementos são estruturados e exibidos na página, mas a aparência visual pode ser estilizada usando CSS (Cascading Style Sheets). Em resumo, o HTML é a linguagem fundamental para a criação de páginas da web. Ele permite a estruturação do conteúdo e a definição de elementos e atributos que são interpretados pelos navegadores para apresentar informações de forma coerente e organizada aos usuários.

O que são Tags

(Voltar ao índice)

Tags são elementos fundamentais na linguagem HTML (HyperText Markup Language) e são usadas para estruturar e definir o conteúdo de uma página da web. Elas são representadas por colchetes angulares (< e >) e indicam ao navegador como deve ser interpretado e exibido cada elemento dentro do documento HTML. As tags são compostas por duas partes principais: a tag de abertura e a tag de fechamento. A tag de abertura é escrita entre colchetes angulares e indica o início de um elemento. Por exemplo, a tag "p" é usada para definir um parágrafo. A tag de fechamento possui uma barra (/) após o sinal de menor (<) e fecha o elemento. Por exemplo, "/p" fecha um parágrafo. Além disso, algumas tags podem ser autossuficientes e não exigem uma tag de fechamento separada. Elas são chamadas de tags vazias ou autonencerrantes. Um exemplo é a tag "img", usada para inserir imagens, que não requer uma tag de fechamento. As tags podem conter atributos adicionais que fornecem informações extras sobre o elemento. Os atributos são inseridos dentro da tag de abertura e consistem em um nome e um valor, separados por um sinal de igual (=). A correta utilização das tags é importante para garantir que o navegador interprete e exiba corretamente o conteúdo da página da web. As tags permitem a estruturação e formatação adequada do conteúdo, além de fornecer informações semânticas para os mecanismos de busca e auxiliar na acessibilidade da página. Em resumo, as tags são elementos essenciais do HTML usados para estruturar e definir o conteúdo de uma página da web. Elas são representadas por colchetes angulares e indicam ao navegador como interpretar e exibir cada elemento dentro do documento HTML.

O que são atributos

(Voltar ao índice)

Os atributos são recursos adicionais que podem ser adicionados às tags HTML (HyperText Markup Language) para fornecer informações extras sobre os elementos. Eles são inseridos dentro da tag de abertura e consistem em um nome e um valor, separados por um sinal de igual (=). Os atributos fornecem instruções e detalhes sobre como um elemento deve ser exibido ou se comportar. Eles podem influenciar características como a aparência visual, o comportamento interativo e as propriedades do elemento. Por exemplo, no caso de um link , o atributo href é usado para especificar o destino do link. O valor do atributo href seria o URL para o qual o link deve direcionar quando clicado. Assim, a tag a href="https://exemplo.com" criaria um link para o site "https://exemplo.com". Outro exemplo é o atributo src usado na tag , que indica o caminho ou URL da imagem que deve ser exibida. A tag "(img src="imagem.jpg") exibiria a imagem "imagem.jpg" no local onde a tag é colocada. Existem muitos atributos diferentes que podem ser usados com várias tags HTML, dependendo do elemento e do efeito desejado. Alguns atributos comuns incluem class para definir classes CSS para estilização, id para identificar elementos exclusivos, style para fornecer estilos inline, alt para fornecer texto alternativo para imagens, entre outros. Os atributos desempenham um papel importante na personalização e no controle do comportamento dos elementos HTML. Eles ajudam a definir as características específicas de cada elemento, fornecendo informações adicionais que os navegadores e outras tecnologias podem usar para interpretar e apresentar corretamente o conteúdo da página. Em resumo, os atributos são recursos adicionais que podem ser usados com as tags HTML para fornecer informações extras sobre os elementos. Eles são inseridos dentro da tag de abertura e consistem em um nome e um valor. Os atributos influenciam a aparência e o comportamento dos elementos e são essenciais para personalizar e controlar como o conteúdo é exibido e interage com os usuários.

Tipografia HTML

(Voltar ao índice)

A tipografia em HTML se refere ao controle e ao estilo do texto em uma página da web. O HTML oferece várias tags e atributos que permitem definir o tamanho, a cor, o estilo e outros aspectos visuais do texto. Para controlar a tipografia, podem ser usadas as tags de cabeçalho "h1" a "h6", que definem diferentes níveis de títulos e têm tamanhos pré-definidos. O "h1" é usado para o título principal, enquanto o "h6" é usado para títulos menores. Além disso, a tag "p" é utilizada para parágrafos de texto. É possível também aplicar formatação inline usando a tag "span", combinada com o atributo style, para definir estilos específicos para partes individuais do texto. Os atributos mais comuns para controlar a tipografia incluem:

  • font-size: define o tamanho da fonte. Pode ser especificado em pixels, pontos, porcentagem ou unidades relativas, como em ou rem.
  • color: define a cor do texto. Pode ser especificado utilizando nomes de cores, valores hexadecimais ou RGB.
  • font-family: especifica a família de fontes a ser utilizada para o texto, como Arial, Verdana, Times New Roman, entre outras. É importante fornecer alternativas em caso de indisponibilidade da fonte desejada.
  • font-weight: define o peso da fonte, como bold (negrito) ou normal (não-negrito).
  • text-decoration: adiciona decoração ao texto, como sublinhado (underline) ou tachado (line-through).
  • text-align: alinha o texto à esquerda (left), à direita (right), ao centro (center) ou justificado (justify).

Listas Ordenadas e Não Ordenadas

(Voltar ao índice)

Em HTML, existem dois tipos principais de listas: listas ordenadas e listas não ordenadas.

As listas ordenadas são utilizadas para apresentar itens em uma sequência específica, geralmente numerados ou com marcadores. Para criar uma lista ordenada, utiliza-se a tag "ol" (Ordered List), e cada item da lista é definido com a tag "li" (List Item). Por exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Já as listas não ordenadas são usadas para exibir itens em uma sequência sem uma ordem específica. Essas listas são representadas por marcadores, como pontos, círculos ou quadrados. Para criar uma lista não ordenada, utiliza-se a tag "ul" (Unordered List), e cada item é definido com a tag "li" (List Item). Por exemplo:

  • Item A
  • Item B
  • Item C

As listas ordenadas e não ordenadas podem ser aninhadas umas dentro das outras para criar uma hierarquia de listas. Por exemplo:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
  3. Item 3

As listas ordenadas e não ordenadas são úteis para estruturar e organizar informações em uma página da web. Elas permitem apresentar itens de forma clara e legível, facilitando a compreensão do conteúdo pelo leitor.

Links

(Voltar ao índice)

Links, também conhecidos como hiperlinks, são elementos essenciais em páginas da web que permitem a navegação entre diferentes documentos, páginas ou seções de um site. Eles são representados pela tag em HTML (HyperText Markup Language). Para criar um link, utiliza-se a tag seguida pelo atributo href, que define o destino do link, ou seja, para onde o usuário será direcionado quando clicar nele. O valor do atributo href pode ser uma URL completa, um caminho relativo ou um identificador de âncora. Por exemplo, para criar um link para o site "https://exemplo.com", utilizamos o seguinte código HTML:

  • Visite o Exemplo
Quando o usuário clica no texto "Visite o Exemplo", ele será redirecionado para o site "https://exemplo.com".

Além do atributo href, a tag pode ter outros atributos opcionais, como: target: especifica como o link será aberto. Por exemplo, target="_blank" abre o link em uma nova janela ou guia do navegador. title: fornece um texto de dica ou descrição que aparece quando o usuário passa o mouse sobre o link. rel: especifica a relação do documento atual com o destino do link, como "nofollow" para indicar que os mecanismos de busca não devem seguir o link. Os links também podem ser criados para seções específicas da mesma página, usando âncoras. Para isso, utiliza-se o atributo href com o identificador da âncora. Por exemplo:

  • Retornar para o índice

Nesse caso, ao clicar no link "Retornar ao índice", a página rolará automaticamente até a seção identificada pelo id="index-page". Os links desempenham um papel fundamental na navegação na web, permitindo que os usuários acessem facilmente diferentes páginas e recursos. Eles são amplamente utilizados para criar menus, direcionar para conteúdo relacionado, vincular documentos e muito mais.
